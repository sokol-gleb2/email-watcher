# 🐍 Script Runner Guide

## ⚙️ Setup

1. **Create virtual environment (if not installe already)**  
   ```bash
   python -m venv venv
   ```

2. **Activate the environment**

```bash
source .venv/bin/activate
```

3. **Install the requirements (if not done already)**

```bash
pip install -r requirements.txt
```

4. **Execute the python file**

```bash
python3 script.py ../server/storage/emails/test.eml ../server/storage/emails/test.json
```
